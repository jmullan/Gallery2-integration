A minimum monitor resolution of 1024x768 is required for the Readme.html file 
to be viewed properly. It also requires javascript be enabled on your web 
browser. If your system does not meet these requirements, you will have to 
install the integration from the Readme.txt file.

All documentation, including the Readme files containing the installation 
instructions, can be found in the docs directory.

The mod files can be found in the mods directory.

The gallery2 and phpBB3 directories contain the required files that need to be 
installed.
